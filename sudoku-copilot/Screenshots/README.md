# Screenshots

Place screenshots here to satisfy the rubric. Recommended filenames:

| File                                    | What to show |
| --------------------------------------- | ------------ |
| `initial_tests.png`                     | Terminal output of `pytest -v` showing all tests pass on the initial code baseline. |
| `copilot_testing_framework.png`         | Your Copilot prompt asking it to set up the testing framework and Copilot's response. |
| `copilot_unique_solution.png`           | Copilot prompt asking to guarantee unique-solution puzzles, and the code it produced. |
| `copilot_top10_localstorage.png`        | Copilot prompt for the Top 10 leaderboard using localStorage, and response. |
| `copilot_alternating_3x3.png`           | Copilot prompt asking to alternate 3×3 grid colors, and the CSS it produced. |
| `copilot_dark_mode.png`                 | Copilot prompt for adding a dark-mode toggle. |
| `copilot_hint_check.png`                | Copilot prompt for the Hint and Check buttons. |
| `copilot_rejected_suggestion.png`       | An example of a Copilot suggestion you REJECTED or modified, with a note explaining why. |
| `final_ui_light.png`                    | Screenshot of the finished game in light mode with the board, controls, timer, and scoreboard. |
| `final_ui_dark.png`                     | Same, in dark mode. |

## How to capture

1. Start the app: `python app.py`.
2. Open http://127.0.0.1:5000
3. Use Windows `Win+Shift+S` (Snipping Tool), macOS `Cmd+Shift+4`, or Linux `gnome-screenshot`.
4. For Copilot conversations, screenshot the VS Code Chat panel including the
   prompt you typed and Copilot's reply.
5. Save each PNG in this folder with the exact filename from the table above.

## Rubric Reminder

Screenshots must show:
- The prompt you sent to Copilot.
- Copilot's response.
- (At least one) an evaluation/rejection of a Copilot suggestion, in either a
  screenshot or a code comment.
